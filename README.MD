  # User Feedback Form — Coding Project 01

  An interactive feedback form built with HTML, CSS, and JavaScript for ISM3232. It captures
  user input and responds in real time using event-based programming — DOM manipulation, event
  delegation, bubbling, mouse events, and input events.

  ## Features

  - Real-time character counter for the comments field
  - Hover tooltips that follow the cursor on each field
  - Form validation that blocks empty submissions with inline error messages
  - Dynamic feedback display that adds each valid submission to the page
  - Event delegation and bubbling handled by listeners on the form
  - `stopPropagation` so background clicks don't trigger form events

  ## Files

  | File | Purpose |
  |------|---------|
  | `index.html` | Form structure: name, email, comments, submit button |
  | `style.css` | Layout, styling, tooltip, and feedback cards |
  | `cp_1.js` | Event listeners and form logic |
